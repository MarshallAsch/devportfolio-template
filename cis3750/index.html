CIS3750 Fall 2017 Lab Demo 2 WireFrame


This website is Team Gallium's (31) Wire frame prototype demo. Click here or the button below to go to the wire framing page. It is a non functional high fidelity prototype. It has been created as part of a community engaged scholarship in partner with Community Living Cambridge for a third year software design course. The goal of this project is to develop a tool to facilitate and engage young adults diagnosed with FASD to lead healthy lives through self-management of their medications.

Our teams role was to build the backed API, which can be found here. Our code is open source and can be found on github here. The front end team that we were parnterned with, Apollo, built an Android mobile application that can be found here, but it is currently not open sourced.

More information on this project can be found on Dan Gillis website here and here.
